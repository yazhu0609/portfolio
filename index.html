病人就醫開單資訊及提醒系統

此作品為大三專題製作的醫療資訊系統。 研究動機為隨著醫療科技的快速進步，遠距醫療及數位化健康管理系統逐漸普及，極大提升了患者的就醫便利性。 然而，隨著人口老化和慢性病患者的增加，醫療系統面臨的壓力也隨之上升。 患者在就醫過程中，仍可能遭遇一些困難，例如就診流程繁瑣、多種紙本單據的管理負擔。 特別是在醫院提供的單據中，檢查單、檢驗單數量龐大，容易造成患者的混淆與誤用。 此外，紙本資料不僅管理不便，也可能導致重要資訊遺失或延誤，進一步影響患者的治療進度及效果。 因此，如何有效簡化就醫流程並協助患者更好地管理就醫資訊，成為目前醫療服務的重要課題。

美食推薦APP

在忙碌且節奏緊湊的大學生活中，許多學生常因午休時間短暫、校內學餐選擇有限，加上用餐人潮眾多， 不得不選擇便利商店作為午餐，或陷入「不知道要吃什麼」的困境，進而忽略了飲食的健康與多樣性。 尤其在講求效率與便利的現代生活中，「吃什麼」反而成為許多人每日都要煩惱的問題。 因此，我們開發這款APP，藉由匯集使用者的餐廳推薦與真實評價，整理並整合學校周邊各類型的餐飲資訊， 協助學生能夠快速搜尋用餐選擇，不僅節省思考時間，更能提升每日飲食的品質與均衡性， 讓學生即使在有限的時間內，也能享有豐富多樣的美食體驗。

掛號病歷表單

此為程式設計課程，實作一個診所出診掛號時需填寫的表單，提供使用者填寫個人病歷資訊的HTML表單， 包含基本資料（如姓名、身分證字號、性別、出生日期、聯絡方式）以及病史、過敏藥物等相關欄位。 使用者填寫完畢後，透過HTTP的POST方法將表單資料傳送至後端的Servlet程式。Servlet接收到表單資料後， 透過JDBC連線至MySQL資料庫，將該筆病人資料插入到預先設計好的資料表中，確保資料完整儲存並具備後續查詢與管理的功能。 資料成功寫入資料庫後，系統會重新導向或回傳一個結果頁面，於HTML網頁上即時顯示該名病患的病歷資訊， 包括其所填寫的所有欄位內容，供使用者確認，達成掛號流程的資料建立與回饋顯示。

電子病歷

此為在電子病歷課程，實作一個病歷資訊管理系統。 採用 ASP.NET MVC 架構，從資料庫設計出發，建立 Patient Table 並對應 ViewModel 與 DBModel， 實作包含資料的新增、查詢、修改與刪除（CRUD）功能。透過 Controller 負責畫面載入與資料處理邏輯， 前端以 HTML 和 Script 製作互動表單，實現從畫面輸入到資料庫儲存的完整流程。

貪食蛇小遊戲

使用軟體為IntelliJ IDEA，語言為Java。 使用者進入遊戲則馬上開始遊戲，畫面主要分為三個區域，左側的畫面上呈現三個小圖示：吃到炸彈則立即結束遊戲、 吃到鞋子則每吃一次加速1.5倍、吃到草莓則會加一分；右側的畫面顯示蛇的長度、生存時間以及分數； 最下方的畫面則分成三個按鈕，由左至右分別是暫停、繼續及離開遊戲。
